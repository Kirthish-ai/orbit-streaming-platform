# ORBIT — Streaming Platform

## Project Structure

```
Orbit/
├── frontend/          # All client-side files
│   ├── index.html
│   ├── style.css
│   └── app.js
│
└── backend/           # Node.js + Express + MongoDB
    ├── server.js
    ├── .env
    ├── package.json
    └── node_modules/
```

## Running the app

```bash
cd backend
node server.js
```

Then open → http://localhost:3001

## Requirements
- Node.js
- MongoDB running on localhost:27017
